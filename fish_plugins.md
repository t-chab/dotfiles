# Useful fisherman plugins list

[fisherman](http://fisherman.sh) is an awesome plugin manager for [fish](http://fish.sh) shell. Here's some useful plugins :

- omf/theme-l
- omf/android-sdk
- anicode
- ansible-completion
- omf/archlinux
- docker-completion
- done
- omf/emacs
- omf/extract
- omf/export
- paths
- omf/pbcopy
- omf/proxy
- omf/ssh
- termcolours
- omf/tmux
- omf/xdg
